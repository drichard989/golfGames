## How To Contribute

1. Open an issue describing the bug or enhancement.
2. Keep changes focused and avoid unrelated refactors.
3. Include clear reproduction and testing notes.
4. For UI changes, include before/after screenshots when possible.

## Development

Run locally from repo root:

```bash
python3 -m http.server 8000
```

